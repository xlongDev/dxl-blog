ECharts 是一个由百度开源的、基于 JavaScript 的数据可视化库，提供多种交互式可视化图表。凭借强大的功能和灵活的配置，ECharts 被广泛应用于数据分析、仪表盘和复杂的业务场景。

## 什么是 ECharts

ECharts（Enterprise Charts）是一个开源、跨平台的可视化库，专注于高性能、交互性强的图表渲染。它的底层基于 HTML5 Canvas 技术，支持运行在各类现代浏览器中，包括移动端。

## 核心功能

### 1. 丰富的图表类型
ECharts 支持多种图表类型，适用于多种场景：

- **基础图表**：折线图、柱状图、饼图、散点图。
- **高级图表**：雷达图、热力图、K 线图。
- **地理可视化**：地图、飞线图。
- **三维可视化**：3D 柱状图、3D 散点图。
- **定制图表**：关系图、桑基图、仪表盘。

### 2. 高度可定制化
通过灵活的配置项，用户可以精细化控制图表的各个部分，包括：

- 图例和标题样式
- 颜色和渐变效果
- 坐标轴和标签格式化
- 动画效果

### 3. 数据驱动
ECharts 采用 JSON 格式的数据结构，通过设置数据项即可动态更新图表内容，轻松实现数据驱动的可视化。

### 4. 响应式和跨平台
ECharts 内置响应式设计，支持在桌面端和移动端的无缝运行，提供一致的用户体验。

### 5. 丰富的交互
支持缩放、拖拽、动态刷新、点击事件等多种交互方式，为用户提供更多操作空间。

## 安装与使用

### 1. 安装 ECharts
可以通过 npm 或 CDN 引入 ECharts：

#### 使用 npm 安装
```bash
npm install echarts
```

#### 使用 CDN 引入
```html
<script src="https://cdn.jsdelivr.net/npm/echarts/dist/echarts.min.js"></script>
```

### 2. 基本使用
创建一个简单的折线图示例：

```html
<!DOCTYPE html>
<html>
<head>
  <script src="https://cdn.jsdelivr.net/npm/echarts/dist/echarts.min.js"></script>
</head>
<body>
  <div id="chart" style="width: 600px; height: 400px;"></div>
  <script>
    const chart = echarts.init(document.getElementById('chart'));
    const option = {
      title: {
        text: '折线图示例'
      },
      xAxis: {
        type: 'category',
        data: ['周一', '周二', '周三', '周四', '周五', '周六', '周日']
      },
      yAxis: {
        type: 'value'
      },
      series: [
        {
          data: [120, 200, 150, 80, 70, 110, 130],
          type: 'line'
        }
      ]
    };
    chart.setOption(option);
  </script>
</body>
</html>
```

### 3. 与框架集成
ECharts 支持与主流框架（如 Vue、React 和 Angular）集成：

#### 在 Vue 中使用
```bash
npm install echarts
```

```javascript
<template>
  <div id="chart" style="width: 600px; height: 400px;"></div>
</template>

<script>
import * as echarts from 'echarts';

export default {
  mounted() {
    const chart = echarts.init(this.$refs.chart);
    const option = {
      title: {
        text: 'Vue 中的 ECharts 示例'
      },
      xAxis: {
        type: 'category',
        data: ['一月', '二月', '三月']
      },
      yAxis: {
        type: 'value'
      },
      series: [
        {
          data: [10, 20, 30],
          type: 'bar'
        }
      ]
    };
    chart.setOption(option);
  }
};
</script>
```
