JavaScript 自 ES2015（即 ES6）开始进入快速发展阶段，陆续引入了许多语法和功能特性，让开发者能够编写出更现代化、可维护性更高的代码。本文将全面解析从 ES2015 到 ES2020 的重要新特性，并结合代码示例进行说明。

## ES2015 (ES6)

### 1. `let` 和 `const`
ES2015 引入了块级作用域的变量声明方式，替代 `var`，提供了更可靠的变量作用域管理。

```javascript
let count = 10;
const PI = 3.14;
// PI = 3.15; // 报错，const 定义的变量不可重新赋值
```

### 2. 箭头函数
箭头函数提供了简洁的函数声明方式，并且不绑定自己的 `this` 值。

```javascript
const add = (a, b) => a + b;
console.log(add(2, 3)); // 输出: 5
```

### 3. 模板字符串
使用反引号定义的字符串支持多行和嵌入表达式。

```javascript
const name = "JavaScript";
console.log(`Hello, ${name}!`); // 输出: Hello, JavaScript!
```

### 4. 解构赋值
解构赋值可以轻松提取对象或数组的值。

```javascript
const [x, y] = [1, 2];
const {name, age} = {name: "John", age: 30};
```

### 5. 默认参数
函数可以为参数设置默认值。

```javascript
function greet(name = "Guest") {
  console.log(`Hello, ${name}!`);
}
greet(); // 输出: Hello, Guest!
```

### 6. 模块化 (`import` 和 `export`)
引入了标准的模块化语法，替代了以前的 `require` 和 `module.exports`。

```javascript
// module.js
export const greet = name => `Hello, ${name}`;

// main.js
import { greet } from "./module.js";
console.log(greet("Alice"));
```

### 7. 类（Class）
引入了类语法，简化了基于原型的继承方式。

```javascript
class Person {
  constructor(name) {
    this.name = name;
  }
  greet() {
    console.log(`Hello, ${this.name}`);
  }
}
const person = new Person("Alice");
person.greet();
```

## ES2016 (ES7)

### 1. 指数操作符 (`**`)
提供了计算幂的语法。

```javascript
console.log(2 ** 3); // 输出: 8
```

### 2. `Array.prototype.includes`
用于判断数组中是否包含某个值。

```javascript
const fruits = ["apple", "banana"];
console.log(fruits.includes("banana")); // 输出: true
```

## ES2017 (ES8)

### 1. `async` / `await`
简化了异步代码的书写方式，使得异步代码更接近同步代码的语义。

```javascript
const fetchData = async () => {
  const response = await fetch("https://api.example.com/data");
  const data = await response.json();
  console.log(data);
};
```

### 2. `Object.entries` 和 `Object.values`
用于遍历对象的键值对或值。

```javascript
const obj = {a: 1, b: 2};
console.log(Object.entries(obj)); // 输出: [["a", 1], ["b", 2]]
console.log(Object.values(obj)); // 输出: [1, 2]
```

## ES2018 (ES9)

### 1. 剩余/扩展运算符（对象）
扩展运算符支持对象解构。

```javascript
const obj = {a: 1, b: 2, c: 3};
const {a, ...rest} = obj;
console.log(rest); // 输出: {b: 2, c: 3}
```

### 2. 正则表达式新增特性
- `dotAll` 模式 (`/s` 修饰符)
- 正则表达式命名捕获组

```javascript
const regex = /(?<year>\d{4})-(?<month>\d{2})-(?<day>\d{2})/;
const {groups: {year, month, day}} = regex.exec("2025-01-21");
console.log(year, month, day); // 输出: 2025 01 21
```

## ES2019 (ES10)

### 1. `Array.prototype.flat` 和 `Array.prototype.flatMap`
简化了多维数组的展开。

```javascript
const arr = [1, [2, [3]]];
console.log(arr.flat(2)); // 输出: [1, 2, 3]
```

### 2. `Object.fromEntries`
将键值对数组转换为对象。

```javascript
const entries = [["a", 1], ["b", 2]];
console.log(Object.fromEntries(entries)); // 输出: {a: 1, b: 2}
```

## ES2020 (ES11)

### 1. 可选链操作符 (`?.`)
简化了深层嵌套属性的访问。

```javascript
const user = {profile: {name: "Alice"}};
console.log(user?.profile?.name); // 输出: Alice
```

### 2. 空值合并操作符 (`??`)
提供了对 `null` 和 `undefined` 的默认值处理。

```javascript
const value = null ?? "default";
console.log(value); // 输出: default
```

### 3. 动态 `import`
支持按需加载模块。

```javascript
const module = await import("./module.js");
module.greet("Alice");
```

### 4. 全局对象 `globalThis`
提供了跨平台统一的全局对象。

```javascript
console.log(globalThis);
```

通过这些新特性，JavaScript 在过去的几年中实现了显著的改进，开发者可以利用它们编写更高效、现代化的代码。如果你有任何疑问或需要进一步的示例，请在评论区留言！
