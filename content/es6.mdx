JavaScript 的发展离不开 ECMAScript 的持续更新。从 ES2015（ES6）开始，JavaScript 引入了大量新特性，使得代码更加现代化、简洁和高效。本篇文章将全面介绍从 ES2015 到 ES2024 的重要特性及其应用。

## ES2015 (ES6)

### 1. `let` 和 `const`
引入块级作用域的变量声明方式。

```javascript
let a = 10;
const b = 20;
```

### 2. 箭头函数
简化函数表达式，且不绑定自己的 `this`。

```javascript
const add = (a, b) => a + b;
```

### 3. 模板字符串
使用反引号和插值表达式。

```javascript
const name = "John";
console.log(`Hello, ${name}!`);
```

### 4. 解构赋值
快速从对象或数组中提取值。

```javascript
const [x, y] = [1, 2];
const { name, age } = { name: "John", age: 30 };
```

### 5. 默认参数

```javascript
function greet(name = "Guest") {
  console.log(`Hello, ${name}`);
}
```

### 6. Rest 和 Spread 操作符

```javascript
const arr = [1, 2, 3];
const newArr = [...arr, 4];

function sum(...args) {
  return args.reduce((a, b) => a + b, 0);
}
```

### 7. 模块化

```javascript
// 导出
export const PI = 3.14;
// 导入
import { PI } from './module';
```

### 8. Promise
用于处理异步操作。

```javascript
new Promise((resolve, reject) => {
  resolve("Success");
}).then(console.log);
```

### 9. 类（Class）

```javascript
class Person {
  constructor(name) {
    this.name = name;
  }

  greet() {
    console.log(`Hello, ${this.name}`);
  }
}
```

## ES2016 (ES7)

### 1. 数组包含操作符 (`Array.prototype.includes`)

```javascript
const arr = [1, 2, 3];
console.log(arr.includes(2)); // true
```

### 2. 幂操作符

```javascript
console.log(2 ** 3); // 8
```

---

## ES2017 (ES8)

### 1. `async/await`

```javascript
async function fetchData() {
  const response = await fetch('https://api.example.com');
  const data = await response.json();
  console.log(data);
}
```

### 2. Object.entries 和 Object.values

```javascript
const obj = { a: 1, b: 2 };
console.log(Object.entries(obj)); // [['a', 1], ['b', 2]]
console.log(Object.values(obj)); // [1, 2]
```

---

## ES2018 (ES9)

### 1. Rest/Spread 操作符用于对象

```javascript
const obj = { a: 1, b: 2, c: 3 };
const { a, ...rest } = obj;
console.log(rest); // { b: 2, c: 3 }
```

### 2. 异步迭代器

```javascript
async function* asyncGenerator() {
  yield "Hello";
  yield "World";
}

for await (const value of asyncGenerator()) {
  console.log(value);
}
```

---

## ES2019 (ES10)

### 1. `flat` 和 `flatMap`

```javascript
const arr = [1, [2, [3]]];
console.log(arr.flat(2)); // [1, 2, 3]

const mapped = arr.flatMap(x => [x, x]);
console.log(mapped);
```

### 2. `Object.fromEntries`

```javascript
const entries = [['a', 1], ['b', 2]];
console.log(Object.fromEntries(entries)); // { a: 1, b: 2 }
```

---

## ES2020 (ES11)

### 1. 可选链操作符

```javascript
const obj = { a: { b: 2 } };
console.log(obj?.a?.b); // 2
```

### 2. 空值合并操作符

```javascript
const foo = null ?? "default";
console.log(foo); // "default"
```

---

## ES2021 (ES12)

### 1. 数字分隔符

```javascript
const largeNumber = 1_000_000;
console.log(largeNumber); // 1000000
```

### 2. `Promise.any`

```javascript
Promise.any([
  Promise.reject("Error"),
  Promise.resolve("Success"),
]).then(console.log); // "Success"
```

---

## ES2022 (ES13)

### 1. 类的私有字段和方法

```javascript
class Counter {
  #count = 0;

  increment() {
    this.#count++;
  }
}
```

---

## ES2023 (ES14)

### 1. `Array.prototype.findLast` 和 `findLastIndex`

```javascript
const arr = [1, 2, 3, 4];
console.log(arr.findLast((x) => x % 2 === 0)); // 4
```
