Rspack 是一款新兴的前端构建工具，由字节跳动开源，旨在取代传统的构建工具（如 Webpack 和 Vite），以更高效、快速的方式进行前端资源的打包和构建。随着现代前端开发的复杂性不断增加，Rspack 应运而生，致力于提升开发体验和构建性能。

## Rspack 简介

Rspack 的目标是为前端开发者提供一个轻量级、易于使用且具有高度扩展性的构建工具。它基于 Rust 编写，借助 Rust 的高性能特性，极大地提升了构建速度，相比传统的构建工具如 Webpack，Rspack 在处理大型项目时具有显著的优势。

### 核心设计理念

1. **高性能**

   - 基于 Rust 实现的并行化处理
   - 智能缓存机制
   - 增量编译优化

2. **兼容性**

   - 支持 Webpack 生态
   - 平滑迁移策略
   - 插件系统兼容

3. **开发体验**
   - 快速的热更新
   - 友好的错误提示
   - 简化的配置体系

```javascript
// 构建性能对比（示例项目）
const buildStats = {
```

#### 性能优化策略

```javascript
// rspack.config.js
module.exports = {
```

### 2. Webpack 生态兼容

Rspack 提供了与 Webpack 配置高度兼容的接口：

```javascript
// rspack.config.js
module.exports = {
  entry: "./src/index.js",
  output: {
    filename: "[name].[contenthash].js",
    path: path.resolve(__dirname, "dist"),
  },
  module: {
    rules: [
      {
        test: /\.jsx?$/,
        use: [
          {
            loader: "babel-loader",
            options: {
              presets: ["@babel/preset-react"],
            },
          },
        ],
      },
    ],
  },
  plugins: [
    new HtmlWebpackPlugin({
      template: "./src/index.html",
    }),
  ],
};
```

### 3. 现代化特性支持

#### 模块联邦

```javascript
// rspack.config.js
const { ModuleFederationPlugin } = require("@rspack/core");

module.exports = {
  plugins: [
    new ModuleFederationPlugin({
      name: "host",
      remotes: {
        app1: "app1@http://localhost:3001/remoteEntry.js",
      },
      shared: ["react", "react-dom"],
    }),
  ],
};
```

#### 资源优化

```javascript
// rspack.config.js
module.exports = {

      chunks: "all",

        vendors: {
          test: /[\\]node_modules[\\]/,

  },

```

## 最佳实践

### 1. React 项目配置

```javascript
// rspack.config.js
const path = require("path");

module.exports = {
  entry: "./src/index.tsx",
  output: {
    path: path.resolve(__dirname, "dist"),
    clean: true,
  },
  module: {
    rules: [
      {
        test: /\.tsx?$/,
        use: "babel-loader",
        exclude: /node_modules/,
      },
      {
        test: /\.css$/,
        use: ["style-loader", "css-loader"],
      },
      {
        test: /\.(png|svg|jpg|jpeg|gif)$/i,
        type: "asset/resource",
      },
    ],
  },
  resolve: {
    extensions: [".tsx", ".ts", ".js"],
    alias: {
      "@": path.resolve(__dirname, "src"),
    },
  },
};
```

### 2. 性能优化配置

```javascript
// rspack.config.js
module.exports = {
  mode: "production",
  optimization: {
    minimize: true,
    moduleIds: "deterministic",
    chunkIds: "deterministic",
    splitChunks: {
      chunks: "all",
      minSize: 20000,
      minChunks: 1,
      cacheGroups: {
        defaultVendors: {
          test: /[\\]node_modules[\\]/,
          priority: -10,
          reuseExistingChunk: true,
        },
        default: {
          minChunks: 2,
          priority: -20,
          reuseExistingChunk: true,
        },
      },
    },
  },
  cache: {
    type: "filesystem",
  },
};
```
