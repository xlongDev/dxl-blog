在前端开发中，数据结构是构建高效、可维护代码的基础。理解常用数据结构不仅能帮助开发者编写更优雅的代码，还能在性能优化、算法设计等方面提供支持。本文将全面介绍前端开发中常用的数据结构及其应用场景。

## 基本数据结构

### 1. 数组（Array）
数组是一种线性数据结构，用于存储有序的元素集合，常用操作包括增删查改。

**特点：**
- 有序、索引访问。
- 支持动态长度（JavaScript 中）。

**常用场景：**
- 数据批量处理（如批量渲染 UI）。
- 排序、筛选等算法操作。

**示例代码：**
```javascript
const array = [1, 2, 3, 4];
array.push(5); // 添加元素
console.log(array[2]); // 访问元素
```

### 2. 链表（Linked List）
链表是一种由节点组成的线性数据结构，每个节点包含数据和指向下一个节点的指针。

**特点：**
- 动态大小。
- 插入和删除操作效率高。

**常用场景：**
- 实现队列或栈。
- 当需要频繁插入/删除操作时。

**示例代码：**
```javascript
class Node {
  constructor(value) {
    this.value = value;
    this.next = null;
  }
}

class LinkedList {
  constructor() {
    this.head = null;
  }

  add(value) {
    const newNode = new Node(value);
    if (!this.head) {
      this.head = newNode;
    } else {
      let current = this.head;
      while (current.next) {
        current = current.next;
      }
      current.next = newNode;
    }
  }
}
```

## 高级数据结构

### 1. 栈（Stack）
栈是一种后进先出（LIFO）的数据结构。

**特点：**
- 只能在一端进行插入和删除。

**常用场景：**
- 表达式求值。
- 浏览器历史记录。

**示例代码：**
```javascript
class Stack {
  constructor() {
    this.items = [];
  }

  push(element) {
    this.items.push(element);
  }

  pop() {
    return this.items.pop();
  }

  peek() {
    return this.items[this.items.length - 1];
  }
}
```

### 2. 队列（Queue）
队列是一种先进先出（FIFO）的数据结构。

**特点：**
- 只能在一端插入，在另一端删除。

**常用场景：**
- 任务调度。
- 动画队列。

**示例代码：**
```javascript
class Queue {
  constructor() {
    this.items = [];
  }

  enqueue(element) {
    this.items.push(element);
  }

  dequeue() {
    return this.items.shift();
  }

  front() {
    return this.items[0];
  }
}
```

### 3. 集合（Set）
集合是一种无序且不重复的元素集合。

**特点：**
- 无重复值。
- 高效的查找和删除。

**常用场景：**
- 去重。
- 交集、并集操作。

**示例代码：**
```javascript
const set = new Set([1, 2, 3]);
set.add(4);
set.delete(2);
console.log(set.has(3));
```

### 4. 字典（Map）
字典是一种键值对存储的数据结构。

**特点：**
- 键值对存储。
- 高效查找。

**常用场景：**
- 缓存。
- 关联数据。

**示例代码：**
```javascript
const map = new Map();
map.set('key1', 'value1');
map.set('key2', 'value2');
console.log(map.get('key1'));
```

### 5. 树（Tree）
树是一种分层结构的非线性数据结构，常见的树有二叉树、二叉搜索树（BST）、红黑树等。

**特点：**
- 层级结构。
- 高效的搜索和插入。

**常用场景：**
- DOM 树。
- 文件系统。

**示例代码（简单二叉树）：**
```javascript
class TreeNode {
  constructor(value) {
    this.value = value;
    this.left = null;
    this.right = null;
  }
}

const root = new TreeNode(10);
root.left = new TreeNode(5);
root.right = new TreeNode(15);
```

### 6. 图（Graph）
图是一种由顶点和边组成的非线性数据结构。

**特点：**
- 用于表示复杂关系。
- 可以是有向图或无向图。

**常用场景：**
- 路径规划。
- 社交网络关系。

**示例代码：**
```javascript
class Graph {
  constructor() {
    this.adjacencyList = {};
  }

  addVertex(vertex) {
    if (!this.adjacencyList[vertex]) {
      this.adjacencyList[vertex] = [];
    }
  }

  addEdge(v1, v2) {
    this.adjacencyList[v1].push(v2);
    this.adjacencyList[v2].push(v1);
  }
}
```

## 总结

在前端开发中，选择合适的数据结构能够显著提升代码的性能和可维护性。了解并熟练应用这些数据结构，将为开发者应对复杂场景提供强大的工具。
