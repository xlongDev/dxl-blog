TypeScript 是 JavaScript 的一个超集，由微软开发并维护。它通过添加静态类型、类、模块等特性，使得 JavaScript 代码更加健壮、可维护和可扩展。本篇文章将全面介绍 TypeScript 的核心概念、语法特性、工具链及其在现代前端开发中的应用。

## 什么是 TypeScript？
TypeScript 是一种开源的编程语言，它是 JavaScript 的超集，意味着所有的 JavaScript 代码都是合法的 TypeScript 代码。TypeScript 的主要目标是增强 JavaScript 的可维护性和开发效率，特别是在大型项目中。

### TypeScript 的核心特性
静态类型检查：TypeScript 引入了静态类型系统，允许开发者在编译时捕获类型错误，从而减少运行时错误。

类和接口：TypeScript 支持面向对象编程的特性，如类、接口、继承等。

模块化：TypeScript 支持 ES6 模块语法，使得代码组织更加清晰。

工具链支持：TypeScript 提供了丰富的工具链支持，如代码格式化、重构、自动补全等。

### TypeScript 基础
1. 类型注解
TypeScript 通过类型注解来定义变量的类型。类型注解可以帮助开发者在编译时捕获类型错误。
```typescript
let count: number = 10;
let name: string = "TypeScript";
let isDone: boolean = false;
```
